## `NO_LT`

``` hol4
Tactical.NO_LT : list_tactic
```

------------------------------------------------------------------------

List-tactic which always fails.

Whatever goal list it is applied to, `NO_LT` always fails with string
`` `NO_LT` ``.

### Failure

Always fails.
