## `DISCH` {#Thm.DISCH}


```
DISCH : (term -> thm -> thm)
```



Discharges an assumption.


    
           A |- t
    --------------------  DISCH u
     A - {u} |- u ==> t
    



### Failure

`DISCH` will fail if `u` is not boolean.

### Comments

The term `u` need not be a hypothesis.  Discharging `u` will remove all
identical and alpha-equivalent hypotheses.
