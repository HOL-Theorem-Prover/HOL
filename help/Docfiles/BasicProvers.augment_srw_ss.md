## `augment_srw_ss`

``` hol4
BasicProvers.augment_srw_ss : ssfrag list -> unit
```

------------------------------------------------------------------------

Augments the "stateful" simpset used by `SRW_TAC` with a list of
`simpset` fragments.

`bossLib.augment_srw_ss` is identical to `BasicProvers.augment_srw_ss`
