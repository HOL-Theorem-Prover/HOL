## `REFL_CONSEQ_CONV` {#ConseqConv.REFL_CONSEQ_CONV}


```
REFL_CONSEQ_CONV : conseq_conv
```



Given a term `t` of type `bool` this consequence conversion returns
the theorem `|- t ==> t`.
