## `rand` {#Term.rand}


```
rand : term -> term
```



Returns the operand from a combination (function application).


If `M` is a combination, i.e., has the form `(t1 t2)`, then
`rand M` returns `t2`.

### Failure

Fails if `M` is not a combination.
