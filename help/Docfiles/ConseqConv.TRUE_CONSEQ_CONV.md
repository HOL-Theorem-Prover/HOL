## `TRUE_CONSEQ_CONV` {#ConseqConv.TRUE_CONSEQ_CONV}


```
TRUE_CONSEQ_CONV : conseq_conv
```



Given a term `t` of type `bool` this consequence conversion returns
the theorem `|- t ==> T`.
