## `TRUE_CONSEQ_CONV`

``` hol4
ConseqConv.TRUE_CONSEQ_CONV : conseq_conv
```

------------------------------------------------------------------------

Given a term `t` of type `bool` this consequence conversion returns the
theorem `|- t ==> T`.
