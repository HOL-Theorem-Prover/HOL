## `singleton_of_list` {#Lib.singleton_of_list}


```
singleton_of_list : 'a list -> 'a
```



Turns a single-element list into a singleton.


`singleton_of_list [x]` returns `x`.

### Failure

Fails if applied to a list that is not of length 1.
