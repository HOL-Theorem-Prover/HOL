## `EXISTS_EQ___CONSEQ_CONV` {#ConseqConv.EXISTS_EQ___CONSEQ_CONV}


```
EXISTS_EQ___CONSEQ_CONV : conseq_conv
```



Given a term of the form `(?x. P x) = (?x. Q x)` this consequence
conversion returns the theorem
`|- (!x. (P x = Q x)) ==> ((?x. P x) = (?x. Q x))`.
