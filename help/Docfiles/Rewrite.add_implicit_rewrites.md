## `add_implicit_rewrites`

``` hol4
Rewrite.add_implicit_rewrites: thm list -> unit
```

------------------------------------------------------------------------

Augments the built-in database of simplifications automatically included
in rewriting.

Used to build up the power of the built-in simplification set.
