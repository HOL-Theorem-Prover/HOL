## `empty_tmset`

``` hol4
Term.empty_tmset : term set
```

------------------------------------------------------------------------

Empty set of terms.

The value `empty_tmset` represents an empty set of terms. The set has a
built-in ordering, which is given by `Term.compare`.

### Comments

Used as a starting point for building sets of terms.
