## `dest_eq` {#boolSyntax.dest_eq}


```
dest_eq : term -> term * term
```



Term destructor for equality.


If `M` is the term `t1 = t2`, then `dest_eq M` returns `(t1, t2)`.

### Failure

Fails if `M` is not an equality.
