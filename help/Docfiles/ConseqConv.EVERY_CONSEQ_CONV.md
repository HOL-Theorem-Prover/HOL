## `EVERY_CONSEQ_CONV`

``` hol4
ConseqConv.EVERY_CONSEQ_CONV : (conseq_conv list -> conseq_conv)
```

------------------------------------------------------------------------

Applies in sequence all the consequence conversions in a given list of
conversions.
