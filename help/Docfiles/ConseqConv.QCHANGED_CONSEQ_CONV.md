## `QCHANGED_CONSEQ_CONV` {#ConseqConv.QCHANGED_CONSEQ_CONV}


```
QCHANGED_CONSEQ_CONV : conseq_conv -> conseq_conv
```



Makes a consequence conversion fail if applying it raises the
`UNCHANGED` exception.
