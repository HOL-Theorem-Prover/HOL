## `QCHANGED_CONSEQ_CONV`

``` hol4
ConseqConv.QCHANGED_CONSEQ_CONV : conseq_conv -> conseq_conv
```

------------------------------------------------------------------------

Makes a consequence conversion fail if applying it raises the
`UNCHANGED` exception.
