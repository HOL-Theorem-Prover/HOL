## `triple`

``` hol4
Lib.triple : 'a -> 'b -> 'c -> 'a * 'b * 'c
```

------------------------------------------------------------------------

Makes three values into a triple.

`triple x y z` returns `(x, y, z)`.

### Failure

Never fails.
